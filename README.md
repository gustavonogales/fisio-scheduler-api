## Description

This project was made with [Nest](https://github.com/nestjs/nest)

## Installation

```bash
$ yarn install

# start postgres container
$ docker-compose up -d

# Run migrations and seed
$ yarn migrate
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
